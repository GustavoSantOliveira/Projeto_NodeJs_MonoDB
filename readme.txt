Projeto foi desenvolvido em:
-> Front-end: HTML+CSS+JS
-> Back-end: Node JS
-> Banco de Dados: mongoBD

-- Ao baixar o projeto, instalar as dependências com o comando "npm install" no prompt.

-- Antes de startar o projeto, conectar com o banco MongoBD através do prompt. Acessando a pasta do MongoDB e executar o 
comando "mongod --dbpath C:\Users\Projeto_NodeJs_MogoDB\data",alterando a rota para a pasta "data" onde se encotra
 o banco de dados do projeto.

-- Após realizar a conexão e manter o cmd aberto, em outro prompt executar o comando "npm start".
